# PROJECT-SRF
Project Aplikasi Web Sistem Reservasi Fasilitas , Membantu dalam memanajemen peminjaman fasilitas dalam lingkungan kampus atau sekolah, dibuat untuk keperluan **Praktik Kerja Lapangan(PKL)**.

## Teknologi yang Digunakan

- **Backend**: Laravel 12
- **Admin Panel**: Filament 5
- **Frontend**: Tailwind CSS v4 + Alpine.js
- **Build Tool**: Vite + Node.js
- **Database**: MySQL

### Prasyarat
- PHP 8.3 atau lebih baru
- Composer
- Node.js 20+
- MySQL
